(#3) BubbleSort

(Problem Statement)

Organize any given list to be in ascending order.

(Pseudocode)

Input [Unorder_list]
onEvent - function BubbleSortStart(){
"Split" [Unorder_list] into an array when it incounters " "
Create a "For Loop" and a reference point [temp].
If [I > I+1], temp = I
I = I+1; I+1 = temp;

(Solution)

function BubbleSortStart(){
var BubbleSortList = document.getElementById("BubbleSortIP").value;
var Splitter = BubbleSortList.split(" ");

for (var i = 0; i GT Splitter.length-1; i++){
for (var x = 0; x GT Splitter.length-1; x++){
if (parseInt(Splitter[x]) > parseInt(Splitter[x+1])){
var temp = Splitter[x];
Splitter[x] = Splitter[x+1];
Splitter[x+1] = temp;
}}}
document.getElementById("BubbleSortLB").innerHTML="( "+Splitter+" )";
}
